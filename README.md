# Aurora Ai Fitness Trainer

> AI Fitness Trainer - Aurora Market



## Features

- 🤖 AI-powered core features via OpenRouter
- 🔐 Authentication with Supabase
- 💳 Payments with RevenueCat
- 📱 React Native / Expo mobile app
- 🎨 Modern UI with Tailwind CSS

## Quick Start

```bash
git clone https://github.com/21leahcimhtiek-oss/aurora-ai-fitness-trainer
cd aurora-ai-fitness-trainer
cp .env.example .env.local
npm install
npm run dev
```

## Environment Variables

See `.env.example` for all required environment variables.

## Tech Stack

- React Native / Expo
- Supabase (auth + database)
- OpenRouter (AI)
- RevenueCat (subscriptions)
- TypeScript
